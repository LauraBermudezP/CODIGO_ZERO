Espacio en memoria para guardar datos que pueden cambiar durante el programa.

Estructura que permite ejecutar código solo si se cumple una condición.
Repite un bloque de código varias veces mientras se cumpla una condición.
Colección ordenada de datos almacenados bajo un solo nombre.
Estructura que agrupa propiedades (datos) y métodos (funciones).

Valor que se pasa a una función para que trabaje con él.

Conjunto de funciones que permite la comunicación entre diferentes programas.
